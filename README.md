# MediScan AI 

Medical image classifier for Chest X-Ray (Pneumonia) and Brain Tumor MRI.

## Live Demo
[mediscan-ai.streamlit.app](https://your-url.streamlit.app)

## Models
- Chest X-Ray: DenseNet121 — Binary (Normal/Pneumonia)
- Brain MRI: EfficientNetV2B3 — 4 class (Glioma/Meningioma/No Tumor/Pituitary)

## Datasets
- [Chest X-Ray](https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia)
- [Brain Tumor MRI](https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset)
